# skills

Claude Code skills for developers who want to think better, not just move faster.

---

## dev-check

A structured thinking partner for engineering decisions and agentic development. Challenges your reasoning, stress-tests designs before you build them, and tracks patterns in how you think over time.

Not a code generator. Not a how-to guide. A check on your judgment.

**Triggers on:** `dev check`, `check my thinking`, `review this design`, `weekly check-in`, `what am I missing`, `push back on this`

---

## Install

```bash
/plugin marketplace add github.com/btyeung/skills
/plugin install dev-check@btyeung-skills
```
